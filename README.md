# AdminTailWind

Laravel 13 + Tailwind Admin Template

---

## About This Project
This is a Laravel-based admin dashboard using Tailwind CSS template.

Built for:
- Admin panel system
- Dashboard UI
- Web application backend

---

## Tech Stack
- Laravel 13
- Tailwind CSS
- Vite
- PHP

---

## Features
- Login & Authentication
- Admin Dashboard UI
- Responsive design
- Modern UI components

---

## Installation

```bash
composer install
npm install
npm run dev
php artisan serve
cp .env.example .env
php artisan key:generate
php artisan migrate
